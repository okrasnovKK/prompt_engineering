# Эксперименты с GigaChat API

Этот проект содержит мои эксперименты с GigaChat API в рамках подготовки к работе с LLM и промпт-инжинирингом.

## Содержание

- `basic_api_call.py` — базовый запрос к GigaChat API
- `prompt_comparison.py` — сравнение разных промптов на тестовом датасете
- `rag_prototype.py` — простой RAG-пайплайн с поиском по документам
- `experiments_log.csv` — таблица с результатами экспериментов
- `data/` — тестовые документы для RAG
